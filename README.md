# What's this?
This repository contains the English adaptation of my bachelor's degree thesis.

The paper presents a rigorous mathematical definition of n-dimensional Perlin noise and its derivatives, as well as an overview of the developed computer system for 3D modeling of generated textures using shaders.

# Where to see?
The text is available at: https://github.com/rayskiy7/generica/blob/main/Thesis.pdf

The resulting texture generation system is available in the release: https://github.com/rayskiy7/generica/releases/download/v1.0.0/system.zip
